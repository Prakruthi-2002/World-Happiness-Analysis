# World Happiness Analysis 2019

## Project Overview
This project performs Exploratory Data Analysis (EDA) on the World Happiness Report 2019 dataset to identify factors influencing happiness across different countries.

## Dataset Information
- Dataset: World Happiness Report 2019
- Records: 156 Countries
- Features: 9 Columns

## Objectives
- Analyze happiness scores across countries
- Identify factors affecting happiness
- Perform data cleaning and inspection
- Create visualizations
- Generate meaningful insights

## Exploratory Data Analysis
### Questions Answered
1. Top 10 happiest countries
2. Bottom 10 happiest countries
3. Countries with highest GDP per capita
4. Countries with highest social support
5. Relationship between GDP and Happiness Score

## Visualizations
- Bar Chart
- Histogram
- Scatter Plot
- Pie Chart
- Heatmap
- Line Chart

## Key Insights
- Finland ranked as the happiest country.
- European countries dominate the top rankings.
- GDP per capita shows a strong positive relationship with happiness.
- Social support significantly influences happiness.
- Healthy life expectancy contributes to higher happiness scores.
